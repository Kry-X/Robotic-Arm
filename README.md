# Gesture Controlled Robotic Arm

## Overview

This project is a Gesture Controlled Robotic Arm developed using ESP32 and servo motors. The system enables touchless control of a robotic arm through hand gestures detected by a webcam. Gesture data is processed on a computer and transmitted wirelessly to the ESP32, which controls the movement of the robotic arm in real time.

## Features

* Real-time hand gesture control
* Wireless communication with ESP32
* Multiple servo motor control
* Touchless human-machine interaction
* Web-based manual control interface
* Educational and low-cost design

## Components Used

* ESP32 Development Board
* Servo Motors(MG996R,SG90)
* Robotic Arm Structure
* Webcam
* Power Supply
* Jumper Wires

## Working Principle

1. A webcam captures hand movements.
2. Computer vision detects hand landmarks and gestures.
3. Gesture information is processed on the computer.
4. Commands are transmitted wirelessly to the ESP32.
5. The ESP32 controls the servo motors.
6. The robotic arm moves according to the detected gestures.

## Applications

* Educational Robotics
* Human-Machine Interaction
* Industrial Automation Concepts
* Research and Development
* Smart Control Systems

## Project Outcome

The project successfully demonstrates touchless robotic arm control using hand gestures. It combines computer vision, wireless communication, and embedded systems to create an interactive robotic solution.
